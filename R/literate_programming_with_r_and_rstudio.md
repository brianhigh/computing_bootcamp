## Learning Objectives

You will learn:

* What literate programming is
* Why literate programming matters
* What tools are available for literate programming
* How to use R and RStudio for literate programming
* What kinds of documents RMarkdown supports
* The basics of formatting documents with RMarkdown
* The basics of using *knitr* to render RMarkdown documents
* How to share or publish RMarkdown and rendered documents
