## Benefits

- Scalability
- Secure (data isn't stored on a laptop or desktop)
- Multiuser Access
- Availability

## Available Options

- Shared database servers
    - MySQL (Crud)
- Dedicated database servers
    - MongoDB
    - Apache Cassandra
    - Others (FileMaker, etc)
