# Software

## FileZilla

- Secure
- Free (and Open Source)
- Works both ON and OFF campus

## Alternatives

- WinSCP
- CyberDuck
- Remote Desktop

# Connection Process

## Off Campus

**SFTP**

1. Get whitelisted by visiting the DEOHS Portal
2. Launch FileZilla 
3. Connect to vector.deohs.washington.edu using SFTP

## On Campus

**SFTP**

1. Launch FileZilla 
2. Connect to vector.deohs.washington.edu using SFTP
